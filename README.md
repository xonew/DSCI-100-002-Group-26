# Alternatives to Heart Disease Diagnosis: A Machine-Learning Approach
This is a course project for 2023S DSCI-100: Introduction to Data Science. It includes a machine learning model fitted using k-nearest neighbours to attempt to diagnose heart disease using cheap and common diagnostic signs, such as resting blood pressure and maximum heart rate. [The final submission can be found here.](https://github.com/xonew/DSCI-100-002-Group-26/blob/main/project_submission.ipynb)
